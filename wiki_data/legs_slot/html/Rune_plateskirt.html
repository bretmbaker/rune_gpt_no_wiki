The rune plateskirt is a plateskirt made out of runite. It requires level 40 Defence to wear. The rune plateskirt is identical in function to rune platelegs, but is 1 kg lighter.

It can be created with the Smithing skill by using 3 runite bars and a hammer on an anvil; this requires level 99 Smithing and yields 225 Smithing experience.

Body type B characters may receive these from the crystal chest.

Along with the rune 2h sword and platelegs, this is the best bar-per-alch item in the game. If smithing for money, rune 2h sword, platelegs, and plateskirt will give the best value for each bar.
