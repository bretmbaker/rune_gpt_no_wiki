The infernal harpoon (or) is an infernal harpoon with a Trailblazer tool ornament kit added onto it, and then changed to the Trailblazer Reloaded League style via the "Change-Style" option. The kit gives no additional bonuses, and is only used to add aesthetics to the harpoon. It can be stored in a tackle box.

Players can select the "Change-Style" option to alternate between the Trailblazer Reloaded variant and the original Trailblazer variant.
