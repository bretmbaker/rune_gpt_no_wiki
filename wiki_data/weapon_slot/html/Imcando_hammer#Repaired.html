The Imcando hammer is a wieldable variant of a hammer mostly used for Smithing or Construction. It can be obtained through smashing barronite deposits on the barronite crusher in the Ruins of Camdozaal after completing the Below Ice Mountain quest.

When first obtained, the hammer will be in a broken state. The broken hammer can be fixed by using it on Ramarno, who will repair it for 1,250 barronite shards. Alternatively, it can be exchanged for 350 barronite shards, or 500 shards for a fixed hammer. Once repaired, it can be used as a hammer, both while equipped and unequipped, in any way that a normal hammer can be used.

The Imcando hammer is especially useful in a few situations:

  • When fighting Tempoross, it will not be destroyed by flames nor be washed away by waves.
  • When doing Mahogany Homes or Wintertodt, it saves an inventory slot if wielded.
  • In Barbarian Assault, it can be used by the Defender to repair the Penance Runner traps. In this case, the Defender will save a small amount of time that is normally spent collecting a hammer from the battlefield.
  • When doing Motherlode Mine, you can repair the wheels without picking up a hammer first.

The hammer can be swapped via the "Swap" option, converting it to its off-hand variant so that it can be equipped in the shield slot. This allows it to be equipped alongside Amy's saw, a mist battlestaff, a dragon harpoon, or a dragon pickaxe when doing the aforementioned activities.

It can be stored in a magic wardrobe in a player's costume room.

It can also be used in a free-to-play world, despite being a members object. Though it can only be equipped in a members' world, if it is already equipped upon logging into a free-to-play world, the hammer retains its usual functionality, regardless of the player's membership status.
