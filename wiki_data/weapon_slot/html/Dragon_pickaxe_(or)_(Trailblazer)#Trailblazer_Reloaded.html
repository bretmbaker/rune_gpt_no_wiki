The dragon pickaxe (or) is a dragon pickaxe with a Trailblazer tool ornament kit added onto it. The kit gives no additional bonuses, and is only used to add aesthetics to the pickaxe.

Players can select the "Change-Style" option to alternate between the Trailblazer Reloaded variant and the original Trailblazer variant.

While the pickaxe is untradeable in its ornamented state, it can be reverted anytime, returning the dragon pickaxe and kit.

Changes[edit | edit source]

Date Changes
18 February 2025
(update)

The item can now be equipped from the Bank menu if using the Trailblazer Reloaded style.

(update)

A "Change-Style" option has been added to allow the item to transform into the Trailblazer Reloaded-style.

Trivia[edit | edit source]

  • The colour of the ornamented dragon pickaxe's handle is dependent on the colour of the player's default boots, and can therefore be changed by paying a visit to Yrsa in Rellekka. The variants are shown below:

  • Brown boots

  • Khaki boots

  • Ashen boots

  • Dark boots

  • Terracotta boots

  • Grey boots
