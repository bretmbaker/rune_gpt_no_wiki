The target teleport scroll is an item that can be read to unlock the Teleport to Target spell, which requires level 85 in Magic to use.

It can be purchased at Justine's stuff for the Last Shopper Standing for 250 Last Man Standing points, or at the Bounty Hunter Store for 500 Bounty Hunter points. There will be a confirmation message when the player attempts to read the scroll.

Dialogue [ edit | edit source ]

The following text is transcluded from Transcript:Target teleport scroll .
Using
  • The 'Teleport to Target' spell requires level 85 Magic, and can be cast only within the Wilderness, while you have a Target. This scroll will be destroyed when it has been read.
  • Are you sure you want to learn the spell?
  • Yes, I want to learn the spell.
    • You have learnt the 'Teleport to Target' spell.
    • (End of dialogue)
  • No, I'll keep the scroll.
    • (End of dialogue)
If already read
  • You have already unlocked the spell.
    The scroll can be sold if you no longer need it.
  • (End of dialogue)
