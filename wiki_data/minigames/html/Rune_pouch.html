The rune pouch is an item that can store 16,000 of three types of runes (except runes found in the Nightmare Zone ). Players can cast spells using the runes stored in the pouch, so a rune pouch saves inventory space. Using a needle , the rune pouch can be upgraded to a divine rune pouch with a thread of Elidinis and level 75 Crafting . Players cannot own more than one rune pouch.

A rune pouch note can be purchased from players, the Grand Exchange , or bought from Justine's stuff for the Last Shopper Standing for 75 Last Man Standing points; this note can be exchanged for a rune pouch at any bank . It can also be purchased from a Slayer master for 750 Slayer reward points .

Players can speak to the magic combat tutor in Lumbridge to toggle an option which allows runes to be placed directly into the rune pouch upon pickup and runecrafting, provided that there is a stack of that rune already in the pouch with enough space in it to hold all of them. Alternatively, this option can be toggled from the Settings interface.

Players can also make and load presets into the pouch by configuring it while in a bank interface. In this mode, players have four load-outs available to customise by both the types and quantities of runes for each preset. Runes withdrawn from the bank while the interface is open will be put into the pouch, if space is available.

If players die while carrying a rune pouch in the Wilderness or in a PvP world , they will lose the rune pouch, unless it has been locked via use of a trouver parchment . It should be noted that the Trouver parchment does not protect runes stored within a Rune pouch upon death.

Changes [ edit | edit source ]

Date Changes
9 April 2025
( update )

The "X" quantity can now be used as a left-click option for depositing and withdrawing runes.

2 April 2025
( update | poll )

The rune pouch user interface was updated to include options for withdrawing and depositing 1, 5, X, and All number of runes.

21 February 2024
( update | poll )
  • The rune pouch can now be configured whilst in a bank interface.
  • Players can now configure four separate loadouts for the rune pouch, allowing for instantly filling the pouch within a bank.
  • Runes in a rune pouch can be deposited into a bank on free-to-play worlds.
21 June 2023
( update )

The runes inside a rune pouch will now stay inside upon reclaiming it in a non- PvP death .

15 February 2023
( update | poll )

Crafted runes can now be added directly to the rune pouch.

13 January 2021
( update )

The Bounty Hunter Reward Shop has now closed, and with it, the ability to buy rune pouches from them. You may still encounter the former on Deadman Worlds, where you will still be able to trade them as before.

23 April 2020
( update )

With the reintroduction of Bounty Hunter , Slayer masters once again no longer sell the rune pouch for coins.

27 January 2020
( update )

Slayer masters now again temporarily sell the rune pouch for 1,200,000 Coins (to non-Ironmen only) due to Bounty Hunter being temporarily disabled.

21 November 2019
( update )

Trouver parchments may now be used to protect the rune pouch.

14 November 2019
( update )

The price of a rune pouch from the Bounty Hunter Shop has decreased from 75 to 15 points.

7 November 2019
( update | poll )

Slayer masters no longer sell the rune pouch for coins, and the slayer reward points cost has been reduced from 1,250 to 750.

24 October 2019
( update )

Slayer masters now temporarily sell the rune pouch for 1,500,000 Coins (to non-Ironmen only) due to upcoming Bounty Hunter changes.

15 November 2018
( update | poll )

Players can now buy a rune pouch from a Slayer master for 1,250 Slayer reward points .

9 August 2018
( update )

Rune pouches can now be taken into Glarial's Tomb and the Koschei fight if they do not contain any runes.

17 August 2017
( update | poll )

Players can now speak to the magic combat tutor to toggle whether they want runes on the ground to automatically be stored in the rune pouch or remain in the inventory when picked up.

24 September 2015
( update )

Using ESC to close interfaces now functions with the rune pouch and bank deposit box interfaces.

Trivia [ edit | edit source ]

  • If a player attempts to store the rune pouch while the storage interface is open, the message " Don't be silly. " appears in the chatbox.
