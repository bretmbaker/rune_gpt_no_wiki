Rogue equipment

Rogue equipment is set of skilling equipment obtained from a rogue's equipment crate , which are obtained from successfully opening the wall safes at the end of the Rogues' Den minigame. As they are untradeable, the set effectively requires level 50 Agility and Thieving to wear.

The set of clothing can be stored in an armour case within the costume room of a player-owned house . Note: Like most untradeable items, the pieces will always be kept on death, except over level 20 Wilderness . These items cannot be reclaimed from Perdu .

While wearing pieces of the rogue armour, there is a chance of pickpocketing twice the loot from an NPC , with the full set guaranteeing double loot with the exception of coin pouches , where the set instead has chance to give you one coin pouch as well as an amount of coins equal to the value of that pouch, rather than just two pouches. Each piece increases the chance of receiving double loot by 15%, unless all five components are worn together, in which case the odds of doubling the loot becomes 100%. [1] The rogue armour pieces are interchangeable for the sake of its set effect, only the number of pieces worn determines the overall activation rate. [2]

Experience drops remain unaffected when the rogue armour set effect doubles the loot of a pickpocket.

The set effect does not affect stealing from chests, stalls, or activities such as Pyramid Plunder or Underwater Agility and Thieving .

Changes [ edit | edit source ]

6 August 2020
( update )

The Rogues' Kit set has been listed on the Equipment Info screen.

( update )

Wearing items from the Rogues set increases the chance of pickpocketing double loot with the full set providing a 100% chance.

Trivia [ edit | edit source ]

  • The level 135 Rogues at the Rogues' Castle appear to be wearing a set of this armour.
  • Some items are unaffected by Rogue's Equipment, such as triangle sandwiches from the construction workers at the Tower of Life .
