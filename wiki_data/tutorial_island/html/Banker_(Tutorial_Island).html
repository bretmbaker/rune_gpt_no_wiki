These bankers run the Tutorial Island branch of the Bank of Gielinor .

Unlike the rest of Gielinor's bankers, attempting to talk to them will open the bank interface instead.

Date Changes

( update )

The NPC was permanently added with the launch of RuneScape 2 .

25 March 2004
( update )

The NPC was added to the RuneScape 2 Beta.
