The crystal tool seed is an elven crystal that can be turned into the crystal axe, crystal harpoon, or crystal pickaxe, using 120 crystal shards and the dragon counterpart on a singing bowl. It can be obtained as a drop from Zalcano.

The crystal tools start off at 10,000 charges and degrades, like other crystal equipment. One charge is depleted for each time an item is obtained using the tool, with some exceptions for particular uses of the axe and pickaxe where the crystal version has no additional benefit (duplicates from items such as the Varrock armour do not deplete an extra charge). The player may add charges with crystal shards, with each shard granting 100 charges, up to a maximum of 20,000 charges. Dismantling a crystal tool will return the dragon equivalent and crystal tool seed to the player and doing so will lose all crystal shard charges that were previously loaded.

Players can also exchange the tool seed for 100 crystal shards by speaking to Amrod in Prifddinas' Hefin Inn.

A dangerous death in PvP causes the crystal tools to revert to their dragon equivalents which will then be dropped for the killer. All crystal shards will be lost as well.

Tool information[edit | edit source]

Players who do not meet the Crafting and Smithing requirements for each tool can talk to Conwenna or Reese to sing the tools for them for an additional 60 crystal shards per tool.
