Awakener's orbs are items dropped by The Leviathan, The Whisperer, Vardorvis, and Duke Sucellus. They are used to access stronger, awakened variants of the four bosses. Apart from having higher stats, the bosses now absorb any stat drains and have new and/or modified mechanics in their fight.

The orb can only be used once the quest Desert Treasure II - The Fallen Empire is completed.[1]

When attempting to use the orb to start an awakened version of the fight, the game will send a prompt asking the player if they would like to do so:

  • Vardorvis: Prompt appears when attempting to climb the rocks leading to the ritual site
  • Duke Sucellus: Prompt appears when attempting to enter the asylum
  • The Leviathan: Prompt appears when attempting to climb the handholds leading to the boss arena
  • The Whisperer: Prompt appears when interacting with her as the Odd Figure in an instance that the player did not receive an orb drop from

Each orb will only persist for one attempt, so the awakened fight should only be started if the player has considered themselves prepared for the challenge, and it will not be returned regardless if the player successfully completes the encounter or not. The on-task boost of the slayer helmet will not work in the fight against the awakened bosses.

While the awakened bosses have a three times better drop rate for unique items, they are not meant to be farmed for such items, due to the cost of the orbs, longer kills and the significant spike in difficulty compared to the boss' normal fights. The main purpose of using orbs is to kill all four awakened bosses for the ancient blood ornament kit, which turns Torva armour into its cosmetic sanguine variant.

Changes[edit | edit source]

Date Changes
3 August 2023
(update)

Awakener's orbs have been added to the collection log. Any orb obtained via the bosses before this update will not appear in the log.

Trivia[edit | edit source]

  • The awakener's orb and the awakened bosses were inspired by Path of Exile's Uber bosses and its item of the same name.[2]
