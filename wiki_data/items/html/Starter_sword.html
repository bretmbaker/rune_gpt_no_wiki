The starter sword is a weapon obtained from a Deadman starter pack, along with the starter bow and staff. It has no requirements to wield, and is intended to be used to help a player get started in a seasonal Deadman Mode server.

Despite being a shortsword, it is classified as a scimitar, with the stats equal to that of a mithril scimitar.

As with all starter weapons, it cannot be used against other players.
