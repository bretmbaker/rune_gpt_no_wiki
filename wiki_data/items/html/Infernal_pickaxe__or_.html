The infernal pickaxe (or) is an infernal pickaxe with a Trailblazer tool ornament kit added onto it. The kit gives no additional bonuses, and is only used to add aesthetics to the pickaxe.

Players can select the "Change-Style" option to change the pickaxe's appearance to the Trailblazer Reloaded variant.

Changes[edit | edit source]

Date Changes
18 February 2025
(update)

The item can now be equipped from the Bank menu if using the Trailblazer Reloaded style.

27 November 2024
(update)

A "Change-Style" option has been added to allow the item to transform into the Trailblazer Reloaded-style.

Trivia[edit | edit source]

  • The colour of the ornamented infernal pickaxe's handle is dependent on the colour of the player's default boots, and can therefore be changed by visiting a makeover service. The variants are shown below:

  • Brown boots

  • Khaki boots

  • Ashen boots

  • Dark boots

  • Terracotta boots

  • Grey boots
