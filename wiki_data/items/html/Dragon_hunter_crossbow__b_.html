The dragon hunter crossbow (b) is a cosmetic variant of the dragon hunter crossbow in black. It can be created by using kbd heads on the crossbow, after claiming the hard tier rewards from the Combat Achievements system via Ghommal by the Warriors' Guild entrance. Doing so will make the dragon hunter crossbow untradable.

Players can dismantle the crossbow if desired, returning the regular crossbow and heads.
