The dragon harpoon (or) is a dragon harpoon with a Trailblazer tool ornament kit added onto it. The kit gives no additional bonuses, and is only used to add aesthetics to the harpoon. It can be stored in a tackle box.

Players can select the "Change-Style" option to alternate between the Trailblazer Reloaded variant and the original Trailblazer variant.

While the harpoon is untradeable in its ornamented state, it can be reverted anytime, returning the dragon harpoon and kit.
