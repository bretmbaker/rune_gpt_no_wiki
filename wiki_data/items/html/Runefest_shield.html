The Runefest shield is a cosmetic kiteshield adorned with the RuneFest dragon insignia, promoting and celebrating the festivities of RuneFest 2017, which took place on the 23rd of September, 2017 at Battersea Evolution Events Centre in Battersea Park, Wandsworth, United Kingdom.[1]

Players can claim one from Diango in Draynor Village via the right-click "Holiday-items" option.

If destroyed, a replacement can be obtained from Diango in Draynor Village or from the toy box of a costume room.

Changes[edit | edit source]

Date Changes
30 November 2022
(update)

Players no longer need to redeem the code runefest17 to unlock the item from Diango, and can claim it via the "Holiday-items" option.

Trivia[edit | edit source]

  • The code to redeem the shield was intended to be revealed for RuneFest attendees and those tuning in to the all-day stream on twitch.tv/runescape on 23 September 2017. However, it was obtained a day earlier than intended due to players successfully guessing the code.
  • The shield is not allowed on Entrana, despite it not having any defensive or offensive stats.
