The Hosidius blueprints are an item purchased from Amy, leader of the Mahogany Homes, for 2,000 Carpenter points. It can be used to permanently unlock a Hosidius-themed house style.

In order to change the house style, players must redecorate with an Estate agent while having the blueprints in their inventory. When the house style is changed to something else, the blueprints are not returned. However, once unlocked, players may always return to the Hosidius theme for just the 5000 coin fee.

When inspecting the blueprints, a message in the chatbox will appear, stating These blueprints appear to be a set of instructions for creating the Hosidius player-owned house theme. Perhaps an Estate Agent can make use of these?
