An egg is an item used mainly in Cooking to make cakes and egg potatoes. Eggs spawn near most chicken coops and searching coops will always yield an egg. The Culinaromancer's Chest in Lumbridge Castle basement sells varying amounts depending on the number of Recipe for Disaster subquests completed.

Chickens can also sporadically lay eggs. When doing so, squawk! will appear above them. The egg laying is related to a tick cycle that is off by default. When a player poisons or envenoms a chicken, the chicken will lay an egg instead of receiving poison damage.

Trivia[edit | edit source]

  • Using an egg on the Chicken Shrine yields the message Nice idea, but nothing interesting happens
  • Using an egg on a Spirit angler will prompt the player to ask "Would you care for an egg in this difficult time?", which is a reference to the television show It's Always Sunny in Philadelphia. The same can be done with a Damp egg but yields different dialogue from the spirit anglers.
