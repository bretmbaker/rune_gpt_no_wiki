The Raging echoes relic hunter (t3) armour set is an item obtained by exchanging a set of items with a Grand Exchange clerk via their right-click "Sets" option and clicking on the appropriate item set within the Item Sets interface.

It is commonly made in order to conveniently sell all four components of the Raging Echoes Relic Hunter (T3) outfit at once.
