Larch blowpipe is an item in the Sailing alpha. It is a two-handed dart weapon that requires 45 Ranged to wield. It can be created with the Fletching skill at level 70 by using a knife on two larch logs. It can fire darts up to adamant. Unlike the toxic blowpipe it requires only darts and does not require charges.
