An oak shield is a shield that requires at least level 10 Defence to be worn. It is created by using a knife on two oak logs. Creating the shield requires level 27 in Fletching, and grants the player 50 experience.

It is used as the base for creating a hard leather shield.

An oak shield spawns on a table in the Ranging Guild.
