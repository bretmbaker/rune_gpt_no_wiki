The chugging barrel is a tradeable item that allows players to consume up to five potions in a single click. It is purchased from the Mixology Rewards shop in a disassembled state with 17,250 Mox resin, 14,000 Aga resin, and 18,600 Lye resin, which are earned from participating in the Mastering Mixology minigame. The barrel is only tradeable in its disassembled state (where it cannot store potions); players must use the 'Assemble' option in order for the barrel to become useable. While the assembled barrel is untradeable, it is possible to dismantle the barrel at any time, although doing so will cause all potions currently stored inside to be lost.

The barrel can be filled by using up to five different potions on the item itself or via the bank loadout interface; a maximum of 100 doses per potion can be stored. Players can set up to four different loadouts in a bank for the barrel, as well as reorder the potions consumed, filling, and emptying the barrel. Potions and barbarian mixes can be stored, but hunter mixes cannot. Reordering potions allows players to consume potions in a specific order, such as Saradomin brews before super restores, allowing for stats to be boosted correctly. Vials will not be consumed after using all the contents inside the vial. When loading from presets, the barrel will pull from whichever interface is opened (i.e, if looking at potion storage when loading, the barrel will pull potions from pot storage). While it may be beneficial for certain players to place their super attack, strength, and defence potions inside the barrel and drink it as if it were a super combat potion, it is recommended to buy super combat potions since they are cheaper than the barrel.

The barrel can only be used within a bank or next to a bank chest. Certain bank interfaces, such as Eniola, do not allow the use of the barrel.

Ultimate Ironmen are able to store potion doses inside the chugging barrel and consume potions from it. However, they cannot set loadouts as it requires the use of a bank interface to do so. It's contents are also preserved even when the barrel is lost or deleted on death, able to be accessed after assembling a new chugging barrel. This allows it to be used for ghost storage, although due to the time investment to reobtain the barrel, this is not very useful.

The chugging barrel cannot be stored in Chambers of Xeric storage if accidentally brought into the raid.

Changes[edit | edit source]

Date Changes
9 October 2024
(update)

An issue was fixed in which the scroll bar would not consistently appear when configuring the chugging barrel within the bank interface.

2 October 2024
(update)
  • The item was renamed from "Pre-pot device" to "Chugging barrel".
  • The chugging barrel can now be used at the Theatre of Blood, Nex, and Mastering Mixology banks.
  • A bug in which extended anti-venom+ potions were not working correctly with the chugging barrel has been fixed.
  • The resin costs for the barrel has been adjusted:
    • Mox resin: 20,700 -> 17,250
    • Aga resin: 13,450 -> 14,000
    • Lye resin: 28,550 -> 18,600

Trivia[edit | edit source]

  • If a player attempts to store the chugging barrel while the storage interface is open, the message Don't be silly. appears in the chatbox.
  • The animation for assembling and disassembling the chugging barrel is that of using a mortar and pestle, but with empty hands. This can be used effectively as a humorous emote with other players.
