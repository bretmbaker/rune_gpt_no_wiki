An old boot is an untradeable, useless item. It cannot be worn, nor can it be used with another old boot or any other item.

Trivia[edit | edit source]

  • When dropped, the old boot's model is unrealistically large, about double the size of a pair of other wearable boots.
  • When you drop the boot, it looks identical to the forlorn boot.
  • If you attack a Bush while hunting that does not have the desired target inside, there is a chance of receiving this item.
  • Despite not being obtainable from digging in the digsite, the old boot can be used on Terry Balando, causing him to say "Ah yes, an old boot. Not really an ancient artefact, is it?". This is likely because the old boot previously could be obtained from the digsite during RuneScape Classic, and its slot in the drop table was incidentally converted to give leather boots instead of the old boot.
