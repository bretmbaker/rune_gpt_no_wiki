The graceful crafting kit is an interface item seen in-game via the shop interface when browsing Worm Tongue's Wares.

When purchased at the shop, which requires 650 termites and a full set of the uncoloured graceful outfit, the kit itself is not given to the player; rather, the outfit is automatically transformed into the Varlamore-themed variant. This process cannot give termites back in exchange for the original graceful outfit.
