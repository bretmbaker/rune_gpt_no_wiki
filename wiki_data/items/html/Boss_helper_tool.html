The boss helper tool is an item available to use on a beta world.

It contains a variety of functions, which include teleporting to bosses directly and loadouts. Beta specific functions can also be accessed, such as switching to the Ruinous Powers prayer book during the Desert Treasure II Rewards Beta.

If lost, another will be placed in the player's inventory when logging into a beta world. If the player's inventory is full, it will be placed on the ground but will be unobtainable; attempting to pick it up will result in it disappearing and a game chat message stating Too late - it's gone!

Debug functions[edit | edit source]

The tool has a right-click "Debug" option which opens an interface with various handy options.

Desert Treasure II Beta[edit | edit source]

During the Desert Treasure II Beta, the following options were available.

1: Teleports[edit | edit source]

Opens a "Boss list", which contains a list of every boss, raid, as well as Ferox Enclave. When clicked on, the tool will teleport players to the relevant area alongside essential entry items (such as ecumenical keys for the God Wars Dungeon, a knife for Sarachnis, or a dark totem for Skotizo).

Upon teleporting, players will also be assigned the relevant slayer task for the boss, if applicable.

If the boss is located in an area which is locked behind a quest, the quest will be autocompleted without its prerequisites. For example, teleporting to The Gauntlet will autocomplete Song of the Elves, but not the rest of the Elf questline.

Note: The teleport feature can only be used up to level 20 Wilderness.

2: All Gear[edit | edit source]

When in proximity to a bank, opens the same interface as the supplies table found in Ferox Enclave.

If players select this option when far enough from a bank, the game chat message states You need to be near a bank to use this feature.

5: Swap Prayerbook[edit | edit source]

Switches between the Ruinous Powers and the standard prayers.

6: Swap Spellbook[edit | edit source]

Presents the same chat options as the "Venerate" option on an occult altar, allowing switching of spellbooks from anywhere.

2024 Betas[edit | edit source]

The betas run during 2024 featured the same options for 1-3, however, had different options for 4-6.

4: Swap Spellbook[edit | edit source]

Presents the same chat options as the "Venerate" option on an occult altar, allowing switching of spellbooks from anywhere.

5: Set Stats[edit | edit source]

When selected, an interface opens asking "Set which skill?", which can be used anywhere. The list shows all skills, as well as the current levels set in parentheses. Setting stats to a highler level will cause an experience drop to appear equivalent to the amount of experience required for the level. Hitpoints cannot be set to below level 10.

6: Unlock Everything[edit | edit source]

When selected, all quests, miniquests, and Achievement Diaries will be fully completed, and the player-owned house will be fully furnished and moved to Rimmington.
