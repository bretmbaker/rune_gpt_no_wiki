Puffer Fish are fish that are found underwater on Fossil Island. They can be caught and held in a player's inventory as the Pufferfish item. They reduce a player's weight by 0.5kg when held, and can be used used via a "Puff" option to restore 15% oxygen to stay underwater for longer.

Using the Puff action without a diving helmet equipped releases the Puffer Fish back into the water, where it will retreat from the player and appear deflated for several minutes before it can be caught again. The Drop action produces the text You release the pufferfish. In this case, it will not be deflated or retreat from the player and can be caught again immediately.


Changes[edit | edit source]

Date Changes
16 May 2019
(update)

In order to prevent players competing for the same pufferfish, the number of pufferfish has been multiplied by seven and divided into seven sets. Each player can only see one random set of pufferfish, meaning that usually, different players will see different pufferfish, reducing competition for pufferfish.
