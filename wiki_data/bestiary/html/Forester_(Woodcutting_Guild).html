Trivia[edit | edit source]

  • An unused version of the Forester was originally added with the release of RuneScape 2. It was presumably meant to be a generic monster found around Kandarin, but cannot be spoken to unlike the Foresters in the Woodcutting Guild. Although it was never added to the game, it can still be found in the game files.
