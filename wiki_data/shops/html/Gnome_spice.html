Gnome Spice is an ingredient used to make spicy sauce and certain gnome foods . It can be bought from Grand Tree Groceries , run by Hudo in the Grand Tree . It will never run out while making Gnome foods.

To create spicy sauce , gnome spice is used with chopped garlic . This uses up the gnome spice.

It can be added to the nest atop Aldarin Mill for an iron pickaxe (you must relog with the spices inside the nest).
