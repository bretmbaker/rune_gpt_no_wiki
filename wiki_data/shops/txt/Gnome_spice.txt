Gnome Spiceis an ingredient used to makespicy sauceand certaingnome foods. It can be bought fromGrand Tree Groceries, run byHudoin theGrand Tree. It will never run out while making Gnome foods.

To createspicy sauce, gnome spice is used withchopped garlic. This uses up the gnome spice.

It can be added to thenestatop Aldarin Mill for an iron pickaxe (you must relog with the spices inside the nest).
