Blistering Barnacles is a music track that is unlocked on Braindeath Island 's mountain.

Date Changes
1 March 2018
( update )

Added unlock hint for this track: 'This track unlocks on Braindeath Island.'

31 October 2005
( update )

Added to game and set to play in northern Braindeath Island.

Trivia [ edit | edit source ]

  • It is possible the name refers to the repeatedly spoken alliteration "billions of blue blistering barnacles!" Captain Haddock of the Tintin series says.
