Changes [ edit | edit source ]

Date Changes
11 January 2023
( update )

The track's unlock area in Modern music area mode has been slightly expanded to the north-east.

27 April 2022
( update )

The track's unlock area in both Modern and Classic music area mode has been extended to new areas at the cost of the ones north-west and south-west of Nardah in Modern mode.

1 March 2018
( update )

Added unlock hint for this track: 'This track unlocks in Nardah.'

5 December 2005
( update )

Added to game and set to play in Nardah.
