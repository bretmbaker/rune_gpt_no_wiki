Cold War/Quick guide
This quest has an in-depth guide .
It contains a more detailed description of dialogue, cutscenes, and storyline.

Details [ edit | edit source ]

Start point Speak to Larry at Ardougne Zoo . Show on map
Official difficulty Intermediate
Official length Medium
Description At the Ardougne Zoo, one of the zookeepers has become a bit preoccupied by penguins. In fact, he's downright paranoid about them. He needs your help to observe their habits on the iceberg in the Lunar Sea. What begins as a peaceful bird-watching expedition opens a new world of espionage and bad musicians.
Requirements
  • 10 Hunter ( not boostable ) ( required to start )
  • 30 Agility ( not boostable ) ( required to start )
  • 30 Crafting ( not boostable ) ( required to start ) [1]
  • 34 Construction ( not boostable ) ( required to start )
  • 15 Thieving ( not boostable ) ( required to start )
  • Access to a crafting table 3
Items required
  • 10 oak planks
  • 10 steel nails
  • Hammer
  • Spade
  • Clockwork or a steel bar to make one
  • Plank
  • Silk
  • Raw cod or the ring of charos (a)
  • Swamp tar
  • 5 feathers (only regular chicken feathers work)
  • Mahogany plank
  • Leather
Recommended
  • 30 Combat level
  • Fast travel to Lumbridge ( Lumbridge teleport ), Ardougne ( Ardougne teleport or Ardougne cloak 1 ) and the player-owned house ( Teleport to House ), Dramen or Lunar staff (as well as a teleport to access the Fairy ring network )
  • Armour and some food
Enemies to defeat 1–3 Icelords (level 51)
Ironman concerns
  • 50 Woodcutting ( boostable ) for the mahogany plank
League notes

Unlocking Desert ✓ will auto-complete the quest
Unlocking Fremennik ✓ will auto-complete the quest

Walkthrough [ edit | edit source ]

Penguin spotting [ edit | edit source ]

Items needed: 10 Oak planks (unnoted), 10 steel nails, a hammer and a spade. (Note: Followers are not allowed.)

  • Talk to Larry by the penguin cage in the Ardougne Zoo . ( 1 ) …
    1 Okay, why not?
  • Talk to Larry again. ( 1 • 1 ) …
    1 Yes, I have all the materials.
    1 Yes

Hello world! [ edit | edit source ]

  • Travel to the dock northeast of Rellekka and speak to Larry ( 1 ) …
    1
    to sail to the Iceberg.
  • Use an oak plank on firm snow patch.
  • Use your spade on, or dig next to, the newly built hut to cover it in snow.
  • Talk to Larry to enter the bird hide and PAY ATTENTION to this cutscene.
    • Note: Take note of the 3 emotes the left penguin is doing (you will need this several times later, you can re-enter the snow bird hide to watch again before the next step if needed). If you moved onto next step, you cannot go back. You have to spend a lot of time guessing later.
    • See the full guide for a reference of emotes.
  • Talk to Larry. ( 1 ) …
    1 That's crazy!
  • Left-click Travel back to Rellekka using the nearby boat. ( 1 ) …
    1 Yes
  • Talk to Larry on the dock.
  • Read the clockwork book.

Clockwork penguin [ edit | edit source ]

Items needed: A regular plank, a steel bar OR a clockwork mechanism, and a piece of silk, access to a Crafting table 3 or better in a POH.

  • Go to your POH and use your Crafting table 3 to turn your steel bar into a clockwork mechanism. ( 3 ) …
    3 Clockwork mechanism
  • Click on the table again and select clockwork toy and then clockwork penguin . ( 4 • 3 ) …
    4 Clockwork toy
    3 Clockwork penguin
  • Return to Ardougne and talk to Larry. ( 1 • 1 ) …
    1 Yes, I have it.
    1 Yes
  • Talk to Larry once on the iceberg. ( 2 • 1 ) …
    2 It looks like a warning message to keep us away.
    1 Yes

Suit up! [ edit | edit source ]

  • Return to Ardougne Zoo once again.
  • Unequip both hands and cape slots and use the tuxedo-time option on Larry.
  • Enter the penguin pen and talk to the penguin. (only one of them has the "talk-to" option)
    • Use the 3 greeting emotes from earlier. Check your quest journal if you have forgotten, the emotes are noted in there.
  • Read the mission report and exit the pen.
  • Talk to Larry to exit the suit.
  • Talk to Larry about the mission report

Incognito [ edit | edit source ]

Items needed: Clockwork suit, mission report, and ring of charos(a) or a raw cod

  • Talk to Larry in Fred the Farmer 's sheep pen in Lumbridge.
    • Use the tuxedo-time option on Larry.
  • Talk to the penguins who are disguised as a sheep.
  • Do the greeting emotes.
  • Exit the penguin suit by talking to Larry.
  • Talk to Larry again so he teleports you to the Zoo. ( 1 ) …
    1 Yes
  • Use the tuxedo-time option on Larry.
  • Enter the penguin pen and talk to penguin. If using ring of charos(a), make sure it is equipped. ( 1 • 2 • 1 ) …
    1 The penguins in Lumbridge refuse to talk to me.
    2 I must have left the outpost before they gave out the phrase.
    1 Sure!
    Otherwise, use raw cod on Penguin. ( 1 • 1 • 1 ) …
    1 The penguins in Lumbridge refuse to talk to me.
    1 Uh, I forgot what the phase was.
    1 Sure!
  • Exit the suit by talking to Larry.
  • Talk to Larry again.

Intelligence [ edit | edit source ]

Item needed: Clockwork suit. Already bring: 5 feathers, Swamp tar, Mahogany plank, Leather.

  • Go back to Fred the Farmer 's sheep pen.
  • Use the tuxedo-time option on Larry.
  • Talk to the sheep penguin.
  • Talk to Larry to exit suit.
  • Talk to Fred the Farmer. ( 3 • 1 ) …
    3 I need to talk to you about penguins.
    1 Bully Fred
    ( ( 4 • 1 ) …
    4 I need to talk to you about penguins.
    1 Bully Fred
    if you have not completed sheep shearer)
    • Run north between the first field of cows and the mill toward the dairy cow icon on the map, and steal a cowbell from a dairy cow (right click) for later.
    • Kill a chicken if you need 5 feathers.
  • Get back into the Penguin suit and talk to the Sheep Penguin. ( 2 ) …
    2 The Farmer is harmless.
    • If you do not have swamp tar already, run to the Lumbridge swamp to grab some.
  • Talk to Larry again to get teleported to the iceberg. ( 1 ) …
    1 Yes
  • Talk to Larry once at the iceberg.

Surprise, surprise [ edit | edit source ]

Items needed: Swamp tar, 5 Feathers, Lumbridge Mission Report, Ardougne Mission Report

  • Use the tuxedo-time option on Larry
  • Talk to KGP Agent north-west of Larry while in the penguin suit.
  • Do the greeting emotes.
  • Talk to Noodle.
  • Talk to Noodle again to receive the ID card and a mission report. ( 1 ) …
    1 Yeah, I got it.
  • Talk to KGP Agent.
  • Click on the avalanche.
  • Enter the first room to the west and talk to KGP Agent.
  • Exit room through the west door and then through the north door.
  • Walk north once outside to the agility course.

Agility course [ edit | edit source ]

If you run out of food any time and do not wish to risk losing your items, you could un-equip the penguin suit, resulting in being caught and stunned, but otherwise returned to where Larry is on the Iceberg.

  • Climb down the steps and into the water.
  • Avoid the moving ice in the water.
  • Climb the stepping stone at the end.
  • Jump across the next stones until you reach an ice platform.
  • After you manage to cross, continue on the path until you come to a shelf with icicles on it. Use the tread-softly option to get across. You have to repeat this several times at each shelf. If you fail, you will be hit taking a small amount of damage.
  • Next, you have to climb up the ice using a 'Cross Ice' option. This can be very annoying, as you lose 2 hitpoints each time you fail, and the failure rate is surprisingly high. If you have only the minimal Agility — level 30, for example — a small amount of food may help.
  • After that, you have a small cutscene of you sliding down a hill.
  • Talk to Agility Instructor .
  • Unequip the penguin suit and talk to Larry.

'Berg bards [ edit | edit source ]

Items needed: Mahogany plank, Leather (not hard leather), and Cowbells

  • Use the tuxedo-time option on Larry and re-enter the outpost.
  • Talk to the KGP Agent by the control panel at the end of the main hall.
  • Talk to Ping or Pong in the east room.
  • Leave the outpost and head back to Larry.
  • Unequip the penguin suit.
  • Use mahogany plank on the leather to make penguin bongos .
  • Use the tuxedo-time option on Larry.
  • Re-enter the outpost.
  • Talk to Ping or Pong again. ( 2 ) …
    2 Yes.
  • Use the Control panel in the main hall.
  • Enter the War Room to the east.

Conclusive evidence [ edit | edit source ]

  • Kill an icicle-wielding Icelord (you will receive a bonus 40 Attack exp upon killing it.) and exit the pen using the door to the west.
  • Use the chasm to the east of the room to return to the surface.
  • Talk to Larry.
  • Congratulations, quest complete!

Rewards [ edit | edit source ]

  • 1 Quest point
  • 2,000 Crafting experience
  • 5,000 Agility experience
  • 1,500 Construction experience
  • The ability to make the penguin suit . To enter the suit again, you must talk to Larry again at the zoo , pier, or Iceberg
  • The ability to use the Penguin agility course
  • Players can also steal more cowbells and make more bongos . The cowbells or the bongos can only be played in a penguin suit.
