Details [ edit | edit source ]

Start point Speak to Thormac at the top floor of the Sorcerer's Tower , south-east of the Ranging Guild . Show on map
Official difficulty Intermediate
Official length Short
Description Thormac the Sorcerer has a hobby involving scorpions. Unfortunately, three of them have escaped and managed to run far away. If you manage to find them all, he may just be able to perform an important service for you.
Requirements
  • 31 Prayer ( not boostable ) ( not required to start )
  • Completion of Alfred Grimhand's Barcrawl
  • The ability to run past level 172 black demons and level 64 poison spiders
Items required
  • Dusty key (obtainable during the quest) or 70 Agility ( boostable ) .
Recommended
  • 40 Combat level
  • Anti-dragon shield or dragonfire shield (highly recommended, see the guide for the alternative strategy if they are unavailable)
  • Antifire potion (provides complete immunity to the dragonfire if used with an anti-dragon or dragonfire shield)
  • Antipoison or superantipoison
  • Armour and some food
  • Stamina or energy potions
  • Fast travel to several locations
    • The Sorcerer's Tower ( Camelot teleport , Ardougne teleport or Combat Bracelet to the Ranged Guild)
    • Taverley Dungeon ( Taverley teleport , Falador teleport or Teleport to House if your house is in Taverley)
    • Barbarian Outpost ( Games Necklace or Barbarian teleport )
    • Edgeville Monastery ( Combat Bracelet or Amulet of Glory to Edgeville)
Enemies to defeat
  • Jailer (level 47) if you need to acquire the dusty key
League notes

Asgarnia ✓ — location requirement
Kandarin ✓ — location requirement

Walkthrough [ edit | edit source ]

Getting started [ edit | edit source ]

The location of the Sorcerer's Tower.
  • Talk to Thormac on the top floor of the Sorcerer's Tower south of Seers' Village . ( 1 • 1 • 2 ) …
    1 What do you need assistance with?
    1 So how would I go about catching them then?
    2 Ok, I will do it then
    ( B L R )
  • Go to Seers Village and speak to a Seer . (Wait until the seer is done checking himself in the mirror.) ( 1 ) …
    1 Your friend Thormac sent me to speak to you.
    Be sure not to skip this step, as the scorpion in Taverly Dungeon is not accessible without first speaking to the seer.
  • Scorpions can be caught in any order and there is no need to return between each catch.
The location of the first Kharid Scorpion.
The location of the second Kharid Scorpion.
The location of the third Kharid Scorpion.

First scorpion [ edit | edit source ]

  • Note: You will need a dusty key if you don't have 70 Agility, make sure to grab it before entering the dungeon.
  • Go into Taverley Dungeon , (South Taverley) and go to the poison spider area.
    • If you need a dusty key head south to the Black Knights' Base and kill the jailer . Use the key on the jail cell and talk to Velrak the explorer . He will give you the key. Alternatively, a dusty key spawns deep in Taverley Dungeon, near the lava eel fishing spot south of the blue dragon area.
  • Directly south of the level 80 Agility shortcut, search the old wall to enter a small room.
  • Use the scorpion cage on the Kharid Scorpion . If you try to pick it up, it will sting you instead.

Second scorpion [ edit | edit source ]

  • Go to the Barbarian Outpost (a games necklace is helpful).
  • Go inside the fenced in area near the entrance to the agility course.
  • Go inside the building in the north-east corner of the area.
  • Use the scorpion cage on the Kharid Scorpion . If you try to pick it up, it will sting you instead.
    • Note: It is possible for the scorpion to be outside of the small room, in the Agility area, if someone has let it out.

Third scorpion [ edit | edit source ]

  • Go to the Monastery between Edgeville and Ice Mountain (a combat bracelet will take you right there).
  • Climb the eastern ladder and go to the east side of the floor.
    • Note: If the Monk stops you and says only members of the order may go upstairs, simply say that you want to join and you will be allowed to go upstairs.
  • Use the scorpion cage on the Kharid Scorpion . If you try to pick it up, it will sting you instead.

Finishing Up [ edit | edit source ]

  • Return the full scorpion cage to Thormac .
  • Congratulations, quest complete!

Rewards [ edit | edit source ]

  • 1 Quest point
  • 6,625 Strength experience
  • The ability to have Thormac turn a battlestaff into a mystic staff for 40,000 coins
